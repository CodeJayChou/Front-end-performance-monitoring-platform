1. 能力一：Dedup（去重能力）⭐核心必补
问题

没有 dedup 会变成：

同一个 JS error 上报 100 次
Resource error 重复爆炸
HTTP retry 造成污染
设计：Fingerprint-based Dedup
export interface DedupStore {
  has(key: string): boolean;
  set(key: string, ttl: number): void;
}
fingerprint 生成（统一）
export function createFingerprint(event: ErrorEvent): string {
  return hash([
    event.source,
    event.message,
    event.context?.url || '',
    event.stack?.[0]?.functionName || ''
  ]);
}
Core Pipeline 插入点
function dedupStep(event: ErrorEvent, store: DedupStore) {
  const key = createFingerprint(event);

  if (store.has(key)) return null;

  store.set(key, 60_000); // 1min window

  return event;
}
2. 能力二：Rate Limit（流控能力）⭐必须有
问题

突发错误（比如 CDN 挂了）会：

打爆 SDK
打爆后端 ingest
造成成本灾难
设计：Token Bucket
export class RateLimiter {
  constructor(
    private capacity: number,
    private refillPerSec: number
  ) {}

  private tokens = this.capacity;
  private last = Date.now();

  allow(): boolean {
    const now = Date.now();
    const delta = (now - this.last) / 1000;

    this.tokens = Math.min(
      this.capacity,
      this.tokens + delta * this.refillPerSec
    );

    this.last = now;

    if (this.tokens < 1) return false;

    this.tokens -= 1;
    return true;
  }
}
Pipeline 接入
function rateLimitStep(event: ErrorEvent, limiter: RateLimiter) {
  return limiter.allow() ? event : null;
}
3. 能力三：Context Normalization（上下文归一）⭐非常关键
问题

跨端最大问题：

web / rn / node context 不一致
字段乱
无法聚合分析
统一 Context Model
export interface ErrorContext {
  url?: string;

  route?: string;

  component?: string;

  action?: string;

  userId?: string;

  traceId?: string;
}
归一化函数
export function normalizeContext(ctx: any): ErrorContext {
  return {
    url: ctx.url || location.href,
    route: ctx.route || location.pathname,
    component: ctx.component || 'unknown',
    action: ctx.action,
    userId: ctx.userId,
    traceId: ctx.traceId,
  };
}
Adapter 必须调用
context: normalizeContext(raw.context)
4. 能力四：Stack Normalization（跨端统一栈）⭐工业级关键
问题

三种 stack：

平台	格式
Web	string stack
RN	hermes stack
Node	V8 stack
统一结构
export interface StackFrame {
  functionName?: string;
  file?: string;
  line?: number;
  col?: number;
  raw?: string;
}
Web parser
export function parseWebStack(stack?: string): StackFrame[] {
  if (!stack) return [];

  return stack.split('\n').map(line => {
    const match = line.match(/at (.+) \((.+):(\d+):(\d+)\)/);

    if (!match) {
      return { raw: line };
    }

    return {
      functionName: match[1],
      file: match[2],
      line: Number(match[3]),
      col: Number(match[4]),
      raw: line,
    };
  });
}
RN / Node 统一 fallback
export function normalizeStack(raw: any): StackFrame[] {
  if (typeof raw === 'string') {
    return parseWebStack(raw);
  }

  return raw || [];
}
5. 能力五：Event Guard（进入 Core 前的“防污染层”）
问题

SDK 最怕：

null event
malformed event
adapter 输出错误结构
Guard 规则
export function eventGuard(event: any): event is ErrorEvent {
  return !!(
    event &&
    event.source &&
    event.message &&
    event.timestamp
  );
}
Pipeline 插入
if (!eventGuard(event)) return null;
6. 完整 Core Pipeline（补齐版）

这是你现在系统的“工业级完整版”：

function process(raw: any, ctx: PlatformContext) {
  let event = adapterRegistry.resolve(raw, ctx);

  if (!event) return;

  // 1. guard
  if (!eventGuard(event)) return;

  // 2. dedup
  event = dedupStep(event, dedupStore);
  if (!event) return;

  // 3. rate limit
  event = rateLimitStep(event, rateLimiter);
  if (!event) return;

  // 4. normalize context
  event.context = normalizeContext(event.context);

  // 5. normalize stack
  event.stack = normalizeStack(event.stack);

  // 6. dispatch
  transport.send(event);
}
